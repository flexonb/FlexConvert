FlexConvert Documentation

All-in-one, offline-first file toolkit.

Overview

FlexConvert runs 100% in your browser. No files are uploaded. Processing is performed using WebAssembly and browser APIs. A tiny backend records anonymous usage counts to improve quality.

Key Features

  • PDF tools: merge, split, compress, rotate, reorder, add/remove pages, watermark, PDF → images
  • Image tools: resize, crop, compress, rotate/flip, format conversion, grayscale, adjustments, text overlay
  • File conversion: DOCX/PPTX/XLSX/TXT → PDF, PDF → DOCX (basic), images → PDF, video/audio convert, extract archives

Privacy

Files never leave your device. Only anonymous usage counts are tracked; no personal data is collected.

See the Privacy & Security page for details.

Shortcuts

  • ⌘K / Ctrl + K: Open Command Palette
  • Drag & drop or paste files into the dropzone
